\section*{Introduction}

This supplemental document accompanies the BAMS manuscript on the Collaborative
Lower Atmospheric Mobile Profiling System (CLAMPS). It provides an extended
visual catalog of representative boundary-layer and mesoscale observation days
from CLAMPS1 and CLAMPS2 deployments. The main article illustrates the facility
with a subset of these cases; this supplement allows readers to browse the full
curated set without expanding the page-limited article.

\subsection*{How to read the figures}

Each case begins with one or more \emph{standard CLAMPS observation} figures:
four-panel summaries of thermodynamic profiles (Panels~A and~B), horizontal
winds (Panel~C), and Doppler-lidar vertical-velocity variance (Panel~D) on a
common 24-hour UTC axis. Auxiliary diagnostic figures follow the case narrative.
Interpretive text is drawn from the online CLAMPS Resources site
(\url{https://eeeeelizzzzz.github.io/CLAMPS_Resources/}).

\subsection*{Related resources}

Figure-reproduction code, an interactive web gallery, and bibliometric analysis
from the review literature collection are archived separately (Zenodo DOI; GitHub repository
\url{https://github.com/eeeeelizzzzz/CLAMPS_Resources}). Animated radar and
satellite loops referenced in the case narratives are available on the live
gallery site.
